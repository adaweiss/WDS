bool Data::checkData ( const string &  bufor)

Metoda dokonuje sprawdzenia poprawnosci otrzymanej wiadomosci, długosc+suma kontrolna.

Suma kontrolna liczona jest jako suma wartości z wszystkich czujników modulo 256. Dodatkowo sprawdzana jest też długość stinga.

RAMKA: string w postaci: x + (4*enc1 w hex) +(4*enc2 w hex)+ 7*foto(0 lub 1)+ sumakontrolna(2*hex)
